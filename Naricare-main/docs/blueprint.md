# **App Name**: HerHealthAI

## Core Features:

- Cycle Tracking: Menstrual cycle tracking: Users log period start dates, and the app stores cycle data for future analysis.
- Period Prediction: AI-driven period prediction: Predicts the start date of the next period, learning from user's logged data. This will act as a reasoning tool which includes mood and physical symptoms.
- Calendar Display: Calendar view: Display tracked cycles and predicted periods in a calendar format.
- Period Alerts: Reminder Notifications: Sends notifications 2 days before the predicted period start date.
- Medication Reminder: Medicine Alarm: set an alarm so user get notified to take medicine with reminders.

## Style Guidelines:

- Primary color: Light lavender (#E6E6FA). Lavender is associated with femininity, serenity, and health.
- Background color: Very light lavender (#F5F5FF). Desaturated for calm feel, consistent with primary color.
- Accent color: Soft blue (#ADD8E6). Analogous to lavender but different enough in brightness and saturation to highlight actionable items.
- Font: 'PT Sans', a humanist sans-serif, for a modern look with some warmth and personality; suitable for both headings and body text
